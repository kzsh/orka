# Supported platforms

## Linux

Fully supported. Pre-built binaries are provided for x86\_64 and ARM64, in both glibc and musl variants. All backends (Docker, Podman, nerdctl, bubblewrap) are available.

## macOS

Not supported. No macOS binary is distributed.

## Windows

Not supported. No Windows binary is distributed. Windows users can try [WSL2](https://learn.microsoft.com/en-us/windows/wsl/install), but this is untested.
